# 🚀 พอร์ตโฟลิโอนักพัฒนาระดับพรีเมียม (Premium Developer Portfolio)

เว็บไซต์พอร์ตโฟลิโอที่เน้นความสวยงาม ทันสมัย และประสิทธิภาพการทำงานระดับสูง สร้างขึ้นด้วยเทคโนโลยีเว็บมาตรฐานที่ได้รับการปรับแต่งมาอย่างดี

## ✨ คุณสมบัติเด่น (Features)

-   **ดีไซน์แบบ Glassmorphism ระดับพรีเมียม**: ใช้พื้นหลังโปร่งใสแบบกระจก (Blurred backgrounds) พร้อมไล่เฉดสีที่ทันสมัยและสะอาดตา
-   **ระบบ Particles พื้นหลัง**: ระบบจำลองอนุภาค (Particle Simulation) บน Canvas ในส่วน Hero ที่เคลื่อนที่อย่างเป็นธรรมชาติ
-   **เอฟเฟกต์ Cursor Glow**: แสงเรืองรองที่จะวิ่งตามตำแหน่งเมาส์ ช่วยสร้างความน่าสนใจและทำให้ UI ดูมีชีวิตชีวา
-   **Infinite Skills Marquee**: แถบแสดงทักษะที่เลื่อนอย่างต่อเนื่องแบบไร้รอยต่อ พร้อมเอฟเฟกต์เรืองแสงตามสีประจำแบรนด์ของเทคโนโลยีนั้นๆ เมื่อนำเมาส์ไปชี้
-   **Project Gallery ระดับมืออาชีพ**: การ์ดแสดงผลงานที่มาพร้อมเอฟเฟกต์เงาสะท้อนแบบกระจก (Glass Reflection) และการขยายตัวที่นุ่มนวล
-   **Certificate Lightbox**: ระบบแสดงใบประกาศนียบัตรที่ดูง่าย พร้อมหน้าต่างแสดงภาพขนาดเต็ม (Lightbox modal)
-   **การปรับแต่งประสิทธิภาพ (Performance Optimized)**: 
    -   ใช้ระบบ `loading="lazy"` เพื่อให้หน้าเว็บโหลดได้รวดเร็ว
    -   ป้องกันการกระตุกของหน้าเว็บ (Zero Layout Shift) ด้วยการกำหนดขนาดภาพที่ชัดเจน
    -   รองรับการแสดงผลทุกหน้าจอ (Responsive Design) ทั้งมือถือ แท็บเล็ต และคอมพิวเตอร์
-   **รองรับ Safari สมบูรณ์**: ปรับแต่ง `-webkit-backdrop-filter` เพื่อให้ผู้ใช้บน iOS และ macOS ได้รับประสบการณ์ที่สวยงามไม่ต่างกัน

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

-   **โครงสร้าง (Structure)**: HTML5 (Semantic HTML)
-   **การตกแต่ง (Styling)**: Vanilla CSS3 (Custom Properties / Micro-animations)
-   **การทำงาน (Interactivity)**: Vanilla JavaScript (ES6+)
-   **ไอคอน (Icons)**: [Devicon](https://devicon.dev/) (Branded SVG Logos)
-   **ฟอนต์ (Fonts)**: Inter (Google Fonts)

## 📥 วิธีใช้งาน (Getting Started)

### 1. สิ่งที่ต้องเตรียม
ใช้เบราว์เซอร์รุ่นปัจจุบัน (Chrome, Firefox, Safari, Edge) เพื่อการแสดงผลที่ดีที่สุด

### 2. การติดตั้งและเปิดใช้งาน
1. ดาวน์โหลดหรือ Clone โปรเจกต์ลงในเครื่อง
2. เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ของคุณ หรือรันผ่าน Local Server (เช่น VS Code Live Server)

## 📂 โครงสร้างโปรเจกต์ (Project Structure)

```text
/
├── index.html     # ไฟล์หลักและส่วนการทำงานของ JavaScript
├── style.css      # ระบบดีไซน์และแอนิเมชันทั้งหมด
├── README.md      # เอกสารประกอบการพัฒนา (ภาษาไทย)
└── [images]       # รูปภาพผลงานและใบประกาศนียบัตรต่างๆ
```
